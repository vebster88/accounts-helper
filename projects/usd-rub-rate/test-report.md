## Наблюдения и замечания

1. **Кэш**: структура кэша соответствует спецификации: `rate`, `source`, `source_name`, `timestamp`, `cached_at`.
2. **Вывод**: формат строго соответствует требованиям. Пометка `(кэш)` добавляется только для stale/кэшированных значений.
3. **Verbose**: диагностика пишется в `stderr`, что соответствует спецификации.
4. **Коды возврата**: `0` при успехе, `1` при ошибке данных, `2` при ошибке CLI.
5. **Безопасность**: используется только HTTPS, User-Agent содержит версию.
